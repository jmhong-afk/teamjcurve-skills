# Skills

지민님의 반복 업무를 Claude Code에서 바로 쓸 수 있도록 정의한 스킬 모음.
매일 밤 11시 크론이 새 스킬을 하나씩 생성해 여기에 쌓는다.

## 폴더 구조

```
skills/
├── README.md              ← 이 파일
├── 콘텐츠/                ← 뉴스레터, 스레드, 교보재 등 콘텐츠 생성 업무
├── 운영/                  ← 멤버 관리, 미팅 노트, 이메일 등 운영 업무
├── 교육/                  ← 워크샵 기획, 커리큘럼, 수강생 관리
└── 데이터/                ← 데이터 정리, 리포트, 분석
```

각 스킬 폴더 구조:
```
[스킬명]/
├── SKILL.md        ← 필수. 역할, 입력, 처리 순서, 예시 프롬프트 포함
├── scripts/        ← 선택. 반복 실행 코드
└── references/     ← 선택. 참고 문서
```

## Claude Code에서 쓰는 법

```
@workspace/skills/콘텐츠/뉴스레터-초안/SKILL.md
```

참조 후 SKILL.md 안의 예시 프롬프트로 바로 시작.

## 팀 공유

이 `skills/` 폴더를 GitHub 레포에 올려두면 팀원 누구나 clone 후 동일하게 사용 가능.
신규 팀원 온보딩 시 스킬 레포 clone이 첫 번째 단계.
