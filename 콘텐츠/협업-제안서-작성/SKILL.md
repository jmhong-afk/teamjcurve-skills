# 협업-제안서-작성

## 역할

기업, 기관, 개인 전문가에게 협업을 제안할 때 설득력 있는 구조화 문서를 생성한다.
"왜 우리와 해야 하는가"에 대한 답이 명확하게 담긴 문서가 필요하기 때문에, AI가 논리 구조와 초안을 잡고 사람이 데이터와 뉘앙스를 보완하는 방식으로 작업 시간을 크게 줄인다.

## 입력

사용자에게 받아야 하는 것:
- **제안 주체 소개**: 우리 조직/프로그램 이름, 하는 일, 규모나 실적 (예: "J-CURVE, AI 교육 프로그램, 코호트 5기 수료 80명")
- **제안 대상**: 어떤 기업/기관인지, 해당 조직의 특성 (예: "스타트업 HR 담당자", "대학교 창업지원단")
- **협업 내용**: 원하는 협력 형태 (예: 강의 파견, 장학금 지원, 채용 연계, 공간 제공, 공동 콘텐츠 제작)
- **상대방 혜택**: 이 협업으로 상대방이 얻는 것 (직접 입력하거나 없으면 AI가 추론)
- **제안 형식**: 이메일 첨부 문서 / 슬라이드 개요 / 단일 페이지 요약 중 선택 (없으면 단일 페이지 요약으로 작성)

## 처리 순서

1. **상대방 입장 파악** — 제안서는 "우리가 원하는 것"이 아니라 "상대방이 얻는 것"부터 시작해야 설득력이 생기기 때문에, 먼저 상대방 관점에서 이 협업의 가치를 정리한다.

2. **구조 설계** — 제안서를 읽는 사람은 바쁘다. 핵심을 앞에 배치하고, 뒤로 갈수록 세부 내용을 담는 역피라미드 구조로 설계한다.

3. **초안 작성** — 각 섹션을 채운다. 과장 표현 없이 구체적인 수치와 사실 중심으로 작성하되, 확인되지 않은 수치는 [확인 필요] 태그를 붙인다.

4. **검토 포인트 목록 생성** — 사람이 반드시 직접 확인/수정해야 하는 항목을 별도로 정리한다. 이 단계가 없으면 초안이 그대로 나가는 사고가 발생한다.

## 예시

**예시 1:**
입력: 제안 주체: J-CURVE (AI 실전 교육, 5기 수료생 80명, 취업률 [확인 필요]%). 제안 대상: 중견 IT 기업 HR팀. 협업 내용: 수료생 채용 연계 + 기업 임직원 AI 리터러시 특강 제공. 형식: 단일 페이지 요약.
출력: J-CURVE-IT기업 채용연계 제안서 (단일 페이지, 역피라미드 구조, 검토 항목 포함)

**예시 2:**
입력: 제안 주체: J-CURVE 5기. 제안 대상: 대학교 창업지원단. 협업 내용: 강의실 무상 사용 + 학생 할인 입학. 협업 기간: 6개월. 형식: 이메일 첨부 문서.
출력: 대학 창업지원단 공간 협업 제안서 (이메일 본문 + 첨부 문서 개요, 상대방 메리트 앞 배치)

## 한계 및 주의사항

- **수치는 직접 확인해야 함**: AI는 취업률, 수강생 만족도, 매출 등 실제 데이터를 모른다. [확인 필요] 태그를 붙여 초안을 내보내며, 사람이 반드시 채워야 한다.
- **업계/기업 최신 정보 반영 불가**: 상대방 조직의 최근 뉴스, 인사 변동, 사업 방향은 검색 없이는 반영할 수 없다. 발송 전 상대방 조직을 직접 리서치할 것.
- **법적 구속력 없음**: 이 스킬이 생성하는 문서는 협업 의향서 수준이며 계약서가 아니다. 실제 계약 조건은 법률 검토가 필요하다.
