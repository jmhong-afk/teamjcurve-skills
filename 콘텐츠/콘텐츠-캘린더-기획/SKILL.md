# 콘텐츠-캘린더-기획

## 역할

뉴스레터, SNS, 커뮤니티 공지 등 반복 발행 채널의 콘텐츠를 월/주 단위로 미리 계획한다.
즉흥 발행보다 사전 기획이 효율적인 이유는, 콘텐츠 간 연결성과 일관된 메시지를 유지하면서 실행 부담을 줄이기 때문이다.
이 스킬은 주제 선정, 발행 일정, 포맷 추천까지 한 번에 처리해 기획 시간을 최소화한다.

## 입력

사용자에게 받아야 하는 것:
- 채널: 어떤 채널인지 (예: 뉴스레터, 인스타그램, 링크드인, 커뮤니티 공지, 카카오채널)
- 기간: 몇 주 또는 몇 달치인지 (예: 4월 한 달, 2주치)
- 브랜드/프로젝트 맥락: 어떤 프로젝트인지, 누구를 타깃으로 하는지 (예: 디지털 노마드 커뮤니티 디놈들, 타깃 - 소규모 창업자/프리랜서)
- 핵심 테마나 우선순위: 이번 기간에 집중할 주제나 프로모션이 있다면 명시 (예: 커뮤니티 신규 멤버 모집, 강의 론칭)
- 발행 빈도: 주 몇 회 또는 월 몇 회 발행할 계획인지

## 처리 순서

1. 기간과 발행 빈도를 기준으로 발행 슬롯을 먼저 확정한다 — 날짜 구조를 먼저 잡아야 주제 배치가 가능하기 때문이다.
2. 핵심 테마를 중심으로 콘텐츠 주제를 생성한다 — 브랜드 맥락과 타깃의 관심사를 교차해야 실제로 읽힐 콘텐츠가 나온다.
3. 각 슬롯에 주제를 배치하고 흐름을 검토한다 — 연속되는 콘텐츠가 서로 연결되는지, 같은 유형이 연달아 반복되지 않는지 확인하기 위해서다.
4. 각 콘텐츠의 포맷과 핵심 메시지(1문장)를 붙인다 — 포맷이 정해져야 실제 제작 시간을 예측할 수 있고, 핵심 메시지는 작성 방향의 기준이 된다.
5. 제작 난이도나 외부 자료가 필요한 항목에 플래그를 달아 사람이 사전 준비할 것을 표시한다.

## 출력 형식

날짜별로 정렬된 표 형식을 기본으로 사용한다. 표가 한 눈에 전체 흐름을 파악하기 가장 쉽기 때문이다.
표 아래에 "제작 주의 항목"을 별도로 정리해 사전 준비가 필요한 것만 뽑아 준다.

```
# [프로젝트명] 콘텐츠 캘린더 — [기간]

## 캘린더

| 날짜 | 채널 | 포맷 | 주제 | 핵심 메시지 | 준비 필요 |
|------|------|------|------|------------|----------|
| 4/1 (화) | 뉴스레터 | 인터뷰 | 디노마드 3년차의 수입 구조 | 월 300만 원을 만든 시스템은 의외로 단순했다 | 인터뷰이 섭외 필요 |
| 4/4 (금) | 인스타 | 카드뉴스 | 노마드 도구 TOP 5 | 이것만 있으면 어디서든 일할 수 있다 | - |
| ... | ... | ... | ... | ... | ... |

## 제작 주의 항목

- [날짜]: [이유] → 사전에 [준비할 것]
```

## 예시

**예시 1:**
입력: 채널=디놈들 뉴스레터, 기간=4월(4주), 타깃=프리랜서/소규모 창업자, 핵심 테마=5월 커뮤니티 오픈 프리홍보, 발행 빈도=주 1회
출력:
```
# 디놈들 콘텐츠 캘린더 — 2026년 4월

## 캘린더

| 날짜 | 채널 | 포맷 | 주제 | 핵심 메시지 | 준비 필요 |
|------|------|------|------|------------|----------|
| 4/7 (화) | 뉴스레터 | 에세이 | 혼자 일한다는 것 | 팀 없이 수익 낸 사람들의 공통점 | - |
| 4/14 (화) | 뉴스레터 | 케이스 스터디 | 월 200만 원 수익 구조 분해 | 레버리지 없이 프리는 한계가 있다 | 사례 인터뷰 필요 |
| 4/21 (화) | 뉴스레터 | 커뮤니티 소개 | 디놈들이 생긴 이유 | 혼자 고민하는 사람들을 위한 공간 | - |
| 4/28 (화) | 뉴스레터 | 모집 공고 | 5월 디놈들 오픈 | 원하는 삶을 원한다면, 구조가 먼저다 | 신청 링크 준비 필요 |

## 제작 주의 항목

- 4/14: 실제 수익 구조 사례 필요 → 인터뷰이 섭외 또는 익명 제보 수집 미리 진행
- 4/28: 신청 폼 또는 랜딩페이지 준비 후 링크 삽입
```

**예시 2:**
입력: 채널=인스타그램, 기간=2주, 타깃=AI 교육 관심 직장인, 핵심 테마=없음(일상적 운영), 발행 빈도=주 3회
출력:
```
# 레카이브 인스타그램 콘텐츠 캘린더 — 2주치

| 날짜 | 포맷 | 주제 | 핵심 메시지 | 준비 필요 |
|------|------|------|------------|----------|
| 4/7 (월) | 카드뉴스 | 직장인이 AI를 못 쓰는 진짜 이유 | 도구 문제가 아니라 습관 문제다 | - |
| 4/9 (수) | 릴스 | AI 프롬프트 30초 꿀팁 | 이렇게 물어봐야 제대로 답이 온다 | 영상 촬영 필요 |
| 4/11 (금) | 텍스트 카드 | 이번 주 배운 것 | 작게 실험하고 빠르게 반복하면 된다 | - |
| 4/14 (월) | 카드뉴스 | ChatGPT vs Claude 실무 비교 | 상황에 따라 다르다, 기준을 알려준다 | - |
| 4/16 (수) | 인터뷰 짤 | 수강생 후기 | 막막했는데 이제 매일 쓴다 | 후기 텍스트 수집 필요 |
| 4/18 (금) | 텍스트 카드 | 다음 주 예고 | - | - |
```

## 한계 및 주의사항

- 실제 트렌드나 업계 이벤트를 자동으로 반영하지 못한다 — AI는 실시간 트렌드를 모르기 때문에, 핵심 날짜(기념일, 업계 행사, 마케팅 시즌)는 사람이 직접 확인하고 보완해야 한다.
- 콘텐츠 퀄리티(문체, 톤, 디자인)는 이 스킬 범위 밖이다 — 주제와 구조만 제공하고, 실제 본문 작성은 뉴스레터-초안, 이메일-초안 등 별도 스킬을 사용한다.
- 인터뷰이 섭외, 영상 촬영, 링크 준비 등 외부 자원이 필요한 항목은 AI가 대신할 수 없다 — "준비 필요" 컬럼을 반드시 사람이 검토해야 한다.
